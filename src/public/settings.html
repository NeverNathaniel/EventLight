Genre Weights

Layer 1 of the preference engine. Each genre you weight (1–5) boosts matching events in Top Picks and the relevance sort.

Learned Signals

Layer 2 — tags and artists picked up from events you mark Interested. These decay over time and boost similar events automatically.

RSS / iCal Feeds

Edit feeds.json without touching code. Disable a feed to skip it on refresh; the URL note flags any that may need verifying.

Web Scrapers
